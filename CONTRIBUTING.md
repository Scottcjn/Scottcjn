# Contributing

Thanks for helping improve this profile repository. Contributions should keep
the public profile clear, accurate, and useful for people learning about the
Elyan Labs ecosystem.

## Local Setup

Clone the repository:

```bash
git clone https://github.com/Scottcjn/Scottcjn.git
cd Scottcjn
```

No build step is required for the current Markdown files.

## Contribution Guidelines

- Keep profile text concise and factual.
- Prefer small pull requests that improve wording, links, or structure.
- Do not add private contact details, credentials, or unreleased information.
- Check that all links are valid before submitting.

## Validation

Run:

```bash
git diff --check
```

For Markdown changes, preview the rendered README where possible.

## Pull Request Checklist

- Summarize what changed.
- Mention any links you verified.
- Link the related issue or bounty, if applicable.
